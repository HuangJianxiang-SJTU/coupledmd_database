# CoupledMD figure source data

This directory contains path-neutral source data for main Figures 1-6 and Supplementary Figures S1-S2. Diagram-only panels are accompanied by release-fact or method-definition tables; quantitative panels have exact plotted source tables.

## Figures 1-4 and Supplementary Figures S1-S2

Figure 1 and Supplementary Figure S1 use the 208-system cohort inventory and receptor-family profiles. Figure 2 uses the audited reduced-release facts rather than a planned full-trajectory archive. Figure 3 contains both the 58-system peptide positive-control cohort and all 208 per-system pocket-availability records. Figure 4 contains one open-fraction record for each of seven adjacent TM interfaces in every included system. Supplementary Figure S2 is accompanied by the gateway method and metric dictionary.

## Figure 5

The replica ledger contains all 624 source-simulation records. The plotted table contains the 618 distinct, quantitatively available replicas from 206 systems. Each plotted replica contributes the 95th-percentile receptor TM-core C-alpha RMSD, the 95th-percentile Galpha interface-region C-alpha RMSD after the same receptor-core fit, and the 5th-percentile initial-contact-retention fraction. The exclusion table identifies three unavailable Gs_8HTI replicas and three Gq_7DWC records excluded as duplicate source identities of Gq_8DWC.

## Figure 6

The pocket mapping contains all 2,168 detected pockets plus the two explicit zero-pocket system records. Panels B and C use 669 GPCR-centered pockets from 201 systems in four classes: orthosteric, extracellular vestibule, TM-core allosteric, and intracellular allosteric. Coupling-interface, other, and receptor-unresolved records remain in the mapping with explicit exclusion reasons. Panel D counts GPCRdb generic positions among orthosteric records with mapped positions. Panel A is a representative structural rendering; its displayed original pocket codes and class labels are listed separately.

No source-server filesystem paths are distributed in these tables.
